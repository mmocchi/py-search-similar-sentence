# py-search-similar-sentence

## installization(by rye)

- install rye

- setup project

```
rye sync
```


## installization(by pip)

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


## usage

```
rye run psss print-similar-sqls --sql <SQL>
```

```
rye run psss print-similar-sqls --help
```
